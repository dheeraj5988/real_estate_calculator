

This is a simple web project that combines a few useful tools into one place. It includes pages like a calculator, investment tool, market info, and some kitchen-related utilities.

## Features

* Basic calculator
* Investment calculation page
* Market information page
* Kitchen utility tools
* Shared styling and scripts across pages

## Tech Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repo
2. Open the folder
3. Run `index.html` in your browser

## Notes

This is a basic frontend project made for practice and learning. It can be improved by adding backend support and better UI.
